# Self-host / run the repo (advanced)

**Most users never need this page.** If you only call the hosted API, use **[Start here](../start-here/README.md)** instead.

This section is for engineers who want to:

- run the Cloudflare Worker API locally,
- contribute to this monorepo, or
- operate a private deployment.

**Warning:** You will deal with Wrangler, Supabase, SQL migrations, and secrets. That is intentional complexity kept out of the default developer path.

Next: [LOCAL_DEV.md](./LOCAL_DEV.md) — minimal variables and stub mode.
